# Fakturka (WinForms)

## Opis
Aplikacja desktopowa napisana w języku C# z wykorzystaniem Windows Forms.  
Projekt edukacyjny stworzony w celu nauki tworzenia aplikacji desktopowych oraz utrwalenia podstaw programowania w C#.

Fakturka pozwala na tworzenie i zarządzanie fakturami sprzedaży i zakupu, automatyczne obliczanie kwot brutto na podstawie stawki VAT oraz filtrowanie i przeglądanie faktur.

## Technologie
- C#
- .NET / .NET Framework
- Windows Forms
- Baza danych Oracle

## Funkcjonalności
- dodawanie faktur sprzedaży i zakupu
- dodawanie pozycji do istniejących faktur
- automatyczne obliczanie kwoty brutto na podstawie kwoty netto i stawki VAT
- prosty system logowania
- soft delete – możliwość archiwizacji faktur i przywracania ich z archiwum
- filtrowanie faktur:
  - po datach (od – do)
  - po numerach faktur
  - po statusie aktywne / archiwalne

## Cel projektu
Projekt został stworzony w celach edukacyjnych, aby:
- utrwalić podstawy programowania obiektowego
- nauczyć się pracy z WinForms
- przećwiczyć logikę biznesową aplikacji finansowej
- obsługę danych oraz filtrowanie i walidację

## Czego się nauczyłem
- tworzenia aplikacji desktopowych w WinForms
- obsługi zdarzeń
- rozdzielania logiki od interfejsu użytkownika
- pracy z danymi finansowymi i obliczeń (VAT, kwoty brutto)
- stosowania architektury warstwowej (Models, Data, Repositories)
- walidacji danych wejściowych i filtrowania danych
